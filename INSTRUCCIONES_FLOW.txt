🔧 PASOS PARA ACTIVAR FLOW.CL:

1️⃣  REGISTRARSE EN FLOW.CL
   
   Para pruebas (Sandbox):
   → Ve a: https://sandbox.flow.cl
   → Crea una cuenta
   → Ve a "Configuración" → "API"
   → Copia tu API Key y Secret Key

   Para producción:
   → Ve a: https://www.flow.cl
   → Sigue el mismo proceso

2️⃣  CONFIGURAR CREDENCIALES

   Edita el archivo .env y reemplaza:
   
   FLOW_API_KEY=your-flow-api-key-here
   FLOW_SECRET_KEY=your-flow-secret-key-here
   
   Con tus credenciales reales de Flow.cl

3️⃣  ACTUALIZAR BASE DE DATOS

   Ejecuta:
   npm run db:push
   
   Esto agregará los nuevos campos para pagos

4️⃣  REINICIAR SERVIDOR

   Detén el servidor (Ctrl+C) y vuelve a iniciarlo:
   npm run dev

💰 FLUJO DE PAGO:

1. Usuario llena formulario → Se crea reserva
2. Usuario hace clic en "Pagar con Flow"
3. Es redirigido a Flow.cl
4. Completa el pago en Flow
5. Flow confirma el pago (webhook automático)
6. Usuario regresa a /payment-result
7. Ve confirmación de pago exitoso

🌐 PASAR A PRODUCCIÓN:

1. Obtén credenciales de producción en https://www.flow.cl
2. Actualiza .env:
   FLOW_API_KEY=tu-key-produccion
   FLOW_SECRET_KEY=tu-secret-produccion
   FLOW_API_URL=https://www.flow.cl/api
   BASE_URL=https://tu-dominio.com

3. Configura el webhook en Flow:
   URL: https://tu-dominio.com/api/flow/confirm
   Método: POST

4. Asegúrate de tener HTTPS configurado
